# Aloecrypt JS

Post-quantum secure messaging for JavaScript and TypeScript.

Same cryptographic core as the [Python library](https://aloecrypt-py.aloecraft.org)
— ML-KEM-768, ML-DSA-65, ChaCha20-Poly1305 — compiled to WebAssembly and
driven through an identical plugin interface.
